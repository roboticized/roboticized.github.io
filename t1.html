Creating and Removing Directories in Linux: Ubuntu/CentOS/OpenSUSE

$ mkdir / $ rmdir

The mkdir command is used to make directories from the command line interface (CLI). The rmdir command is used to remove directories.
By defualt these both commands create and remove empty directories but with the help of command options non-empty and nested directories can also be created and deleted respectively.

Syntax

$ mkdir/rmdir [OPTIONS]... <DIRECTORY_NAME>...

mkdir examples

  • To make an empty directory.
    • $ mkdir videos
  • To make a parent directories, nested directories with -p, --parent option.
    • $ mkdir -p music/hindi_songs 
    $ mkdir --parent music/hindi_songs

rmdir examples

  • To remove an empty directory.
    • $ rmdir videos
  • To remove parent directories, nested directories with -p, --parent option.
    • $ rmdir -p music/hindi_songs 
    $ rmdir --parent music/hindi_songs
